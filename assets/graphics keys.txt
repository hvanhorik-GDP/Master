W - Select object World (camera)
G - Select objects Group 
L - Select objects Lights
M - Select objects Models

TAB - Tab to next item
TAB - Select next invisible item

Shift- - Shift key (reverse)
R - Adjust Object colour R
G - Adjust Object colour Group
B - Adjust Object colour B

Shift- - Shift key (reverse)
X - Move object X (sideways)
Y - Move object Y (up)
Z - Move object Z (back)

Left-Arrow - Rotate object X
Right-Arrow - Rotate object Z

Shift- - Shift key (reverse)
S - scale object
